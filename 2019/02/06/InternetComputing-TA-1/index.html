Step 2 : Installation of Package Control



The simplest method of installation is through the Sublime Text console.

Otherwise you can click the View > Show Console menu.


Once open, paste the appropriate Python code for your version of Sublime Text into the console.


This code creates the Installed Packages folder for you, and then downloads the Package Control.sublime-package into it.


Step 3 : Installation all kinds of useful package.


Once you finish installing the package control, you can check the upper-bar whether you can find Sublime Text > Preference > Package Control.


If you find it, you are successfully download it !


From there, you can simply type in ctrl+shift+p on Windows, or cmd+shift+p on a Mac to open the Command Palette.

Once the Command Palette is open, you can then type Install Package, to install a plugin from the Package Control website. You can also type Add Repository, which allows you to add plugins that are not hosted on the Package Control website, by simply using their repository url. And finally, Remove Package, will let you uninstall a plugin from your system by removing it, as well as any assets from the Sublime packages directory.
